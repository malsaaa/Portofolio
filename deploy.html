Langkah 1: Siapkan Proyek Web

Buat folder di komputer untuk proyek dan buat file HTML, CSS, dan JavaScript sesuai kebutuhan.

my-website/
    ├── index.html
    ├── styles.css
    └── script.js

Langkah 2: Buat Repository di GitHub

  1. Masuk ke Akun GitHub.
  2. Buat Repository Baru dengan nama (misalnya, my-website).
  3. Pilih opsi Public.
  4. Klik tombol Create repository.

Langkah 3: Upload Proyek ke GitHub

  1. Clone Repository:
    2. git clone https://github.com/username/my-website.git
  2. Pindah ke Direktori Proyek:
    3. cd my-website
  3. Copy File Proyek ke Repository.
  4. Lakukan Commit dan Push:
    5. git add .
    git commit -m "Initial commit"
    git push origin main

Langkah 4: Aktifkan GitHub Pages

  1. Buka Pengaturan Repository di GitHub.
  2. Klik pada tab Settings.
  3. Scroll ke Bagian Pages.
  4. Pilih Sumber untuk GitHub Pages: main branch.
  5. Klik Save.

Dapatkan URL untuk Website:

https://username.github.io/my-website/

Langkah 5: Cek Website

Buka URL yang telah disediakan dan dapat melihat website yang telah dideploy.
