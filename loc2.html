Penjelasan Membuat Get Location:

1. Struktur HTML

Buat file index.html, tambahkan judul, dan buat elemen <div> sebagai wadah utama.

2. Tombol "Get Location"

Tambahkan tombol <button> dengan atribut onclick="getLocation()" untuk memicu fungsi JavaScript saat diklik.

3. Wadah Lokasi

Tambahkan <div id="location"> untuk menampilkan latitude, longitude, dan akurasi dari posisi pengguna.

4. CSS Tampilan

Buat file style.css atau gunakan <style> untuk memperindah tampilan tombol dan area lokasi.

5. JavaScript Geolocation

Buat fungsi getLocation() untuk memeriksa dan mengambil lokasi pengguna menggunakan Geolocation API, serta menampilkan hasil atau pesan error.
